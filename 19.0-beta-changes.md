# OpenStack CI Framework - Hardcoded Version Inventory

It was decided to create a new branch for ci-framework and ci-framework-jobs. 19.0-beta name was chosen to make possible version changes to be handled simply as parameters and arguments. We need to clean up all hardcoded version strings, release names, frN, and configuration defaults throughout both repositories.


**Repository:** [openstack-k8s-operators/ci-framework (19.0-beta)](https://github.com/openstack-k8s-operators/ci-framework/tree/19.0-beta)
